Клиент для работы с API сервиса [amoCRM](https://www.amocrm.ru/)

## Установка

### Через composer:

```bash
$ composer require "dotzero/amocrm:dev-master"
```

## Тестирование

Для начала установить `--dev` зависимости. После чего запустить:

```bash
$ vendor/bin/phpunit
```
